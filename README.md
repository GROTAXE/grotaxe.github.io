# Grotaxe

Official Grotaxe project hub — portfolio, experiments, product pages, and public entry points for independent systems.

## Overview

This repository powers the main public GitHub Pages site for **Grotaxe**.

It serves as a central hub for:
- project overviews
- public landing pages
- architecture summaries
- frontend entry points
- links to source repositories
- references to external deployment targets, including TON Sites

The goal is to keep the public surface of the ecosystem clear, transparent, and easy to navigate.

## Main role of this repository

`grotaxe.github.io` is the umbrella landing space for the broader Grotaxe ecosystem.

It is intended to present:
- what projects are being developed
- where each project lives
- how each public frontend or access point is exposed

This repository is **not** the backend source of individual products.  
Each serious product should keep its own dedicated repository and release structure.
