# Grotaxe GitHub Pages Hub

Static multi-page landing site for the Grotaxe personal brand. The site acts as a central hub for technical positioning, project entry points, repository context, and public contact surfaces.

## Structure

- `index.html` - central hub page with About, Projects, Links, and Contacts sections
- `assets/css/style.css` - shared visual system, layout, theming, and responsive styles
- `assets/js/main.js` - lightweight JS for theme toggle, mobile navigation, reveal states, and active section highlighting
- `pages/links.html` - overview page tied to the repository README and future documentation links
- `pages/contacts.html` - dedicated contact surface with clean external links
- `projects/ton-tranclucator/index.html` - placeholder project page for TON Tranclucator

## Sections

- `About` presents Grotaxe as an independent systems builder working across embedded, infrastructure, interface, and protocol layers.
- `Projects` is a scalable showcase area for future device, self-hosted, Telegram, TON, and local AI system pages.
- `Links` provides a readable overview layer and a bridge to repository documentation.
- `Contacts` keeps public channels visible without turning the hub into a social directory.

## GitHub Pages Deployment

This repository is build-free and GitHub Pages friendly. Push the files as-is and publish from the repository root or the default Pages branch configuration.

## Notes

- Dark mode is the primary presentation.
- Light mode is included as an optional alternate theme.
- Replace placeholder profile and repository URLs if the final public handles differ.
