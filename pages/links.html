Summary

What the hub is

The Grotaxe hub is a static, build-free GitHub Pages site meant to present a technical identity rather than a resume. It frames projects around system design, self-hosted infrastructure, embedded logic, operator interfaces, protocol concepts, and long-term product architecture.

The homepage is intentionally concise. This page carries the overview layer so the landing experience can stay sharp while the repository still offers enough context for readers, collaborators, and future docs.

Structure

Current repository map

  • Hub homepage `index.html` anchors the site around About, Projects, Links, and Contacts.
  • Shared surface layer `assets/css/style.css` and `assets/js/main.js` keep the design and behavior consistent.
  • Dedicated subpages `pages/links.html`, `pages/contacts.html`, and project stubs are ready for future growth.
